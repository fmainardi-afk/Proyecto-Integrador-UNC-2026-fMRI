# Proyecto-Integrador-UNC-2026-fMRI
Repositorio que contiene los scripts desarrollados en el marco del Proyecto Integrador titulado “Optimización de parámetros y análisis de imágenes de resonancia magnética funcional para investigación en neurociencias”, realizado en la Universidad Nacional de Córdoba (Córdoba, Argentina, 2026).
Incluye herramientas para la implementación y automatización de los softwares MRIQC, CAT12, fMRIPrep y SPM, así como scripts para la realización de análisis de conectividad funcional. 
El código fue implementado en Python y MATLAB, de acuerdo con los requerimientos de cada entorno.
